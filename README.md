# تعلم وربح 🇲🇦

**تطبيق مغربي لمحاربة الفقر عبر تعليم الحرف - من تطوير Nourddine Abatal**

## 📱 وش هو هاد التطبيق؟

تطبيق كيعلم الشباب المغاربة حرفة مربحة في 7 أيام. منين يتعلم، يبيع المنتوج فـ "سوق الحومة". الهدف: محاربة الفقر.

## ⭐ المميزات

- 🎓 تعلم حرفة في 7 أيام بالفيديو والصوت
- 🛒 سوق محلي لبيع المنتوجات
- 💰 ربح فلوس من المبيعات
- 🗣️ واجهة صوتية كاملة (Zero Text) للأميين
- 📱 أوفلاين - خدمة بلا أنترنت
- 🤖 ذكاء اصطناعي لتقييم جودة المنتوجات
- 💳 اشتراك 29 درهم/شهر عبر Google Play

## 🚀 كيفاش يخدم التطبيق؟

1. **تعلم**: اختار حرفة واتبع الفيديوهات
2. **صنع**: طبق اللي تعلمتو وصنع منتوج
3. **صور**: صور المنتوج ديالك
4. **بيع**: حط المنتوج فـ السوق
5. **ربح**: شوف فلوسك كبر

## 🛠️ التقنيات المستخدمة

- **Flutter** للواجهات
- **Firebase** للتخزين السحابي
- **Google Play Billing** للدفع
- **Whisper + Gemini** للذكاء الاصطناعي
